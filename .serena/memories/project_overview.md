Inkdrop plugin project named short-link. Purpose: replace Markdown link URLs in the editor with a short placeholder emoji via CodeMirror marks. Tech stack: JavaScript with Babel-style module syntax, React class component for modal/layout registration, Inkdrop plugin API, CodeMirror 5, event-kit CompositeDisposable. Structure: lib/short-link.js is plugin entrypoint; lib/short-link-message-dialog.js contains the registered React component and editor behavior; menus/, keymaps/, styles/ contain plugin integration assets; package.json declares Inkdrop engine compatibility.
